# 📊 阿西代码统计工具 - 项目概览

## 🎯 一句话介绍
自动化统计 Git 仓库代码量的全栈工具，支持批量分析、分支选择、可视化展示。

---

## 🚀 三步上手

```bash
# 1. 安装依赖
cd frontend && npm install && cd ../backend && npm install && cd ..

# 2. 启动服务
./start.sh

# 3. 打开浏览器
http://localhost:5380
```

详细步骤：[QUICKSTART.md](QUICKSTART.md)

## 💡 快速命令

### 本地开发
```bash
# 启动
./start.sh

# 前端开发（热重载）
cd frontend && npm run dev

# 后端开发
cd backend && npm start
```

### 打包构建
```bash
# 构建前端
cd frontend && npm run build

# 产物在：frontend/dist/
```

### 生产部署
```bash
# 使用 PM2
npm install -g pm2
cd backend
pm2 start server.js --name "collect-code"

# 前端用 Nginx 托管 frontend/dist/
```
